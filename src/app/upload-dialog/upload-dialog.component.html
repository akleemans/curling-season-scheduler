Upload doodle CSV

  1. Export Doodle/Nuudel as Excel
  2. Add "Skip-skill": Open the Excel and after each name (in the same cell), enter a number in brackets, indicating the skip level:
    • (2) = Yes, can be skip
    • (1) = Could be skip if necessary
    • (0) = No, can not be skip
    Example: 
      Max Muster (0)
  Hanna Hagen (2)
  3. Click "Save as", choose "CSV UTF-8 (Comma delimited) (*.csv)"
